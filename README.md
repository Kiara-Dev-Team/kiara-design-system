## ğŸš¦ Getting Started

### Prerequisites

- Node.js (latest LTS version recommended)
- npm, pnpm, or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Kiara-Dev-Team/kiara-design-system.git

# Navigate to the project
cd kiara-design-system

# Install dependencies
npm install
# or
pnpm install
```

### Development

```bash
# Start the development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. 

### Build

```bash
# Create a production build
npm run build
# or
pnpm build

# Start the production server
npm start
# or
pnpm start
```
